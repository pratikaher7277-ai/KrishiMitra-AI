# KrishiMitra AI 🌱
An AI-powered smart farming companion designed to help farmers take the right action at the right time—from sowing to selling.

## 🚀 Core Features
- **Smart Crop Journey:** Interactive questionnaire that maps out personalized, stage-by-stage farming roadmaps and crop guidelines.
- **Krishi Doctor AI:** Instant crop disease identification and treatment care tips via image uploads powered by Gemini 2.5 Flash.
- **MarketMitra AI:** Live crop quality grading, demand forecasts, and estimated revenue calculators to optimize market selling decisions.

## 🛠️ Tech Stack
- **Backend Framework:** Python (Flask)
- **AI Integration:** Google Gemini AI Studio (`gemini-2.5-flash` model)
- **Frontend Architecture:** HTML5, CSS3 (Modern, Responsive Dashboard Interface)
